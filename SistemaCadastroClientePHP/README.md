# 📋 Sistema de Cadastro de Cliente

Projeto desenvolvido para a disciplina de **Programação Web II**.

## 🎯 Objetivo
Criar um sistema simples de cadastro utilizando:
- Formulário HTML  
- Processamento com PHP  
- Validações de dados  

## ⚙️ Funcionalidades
- Cadastro de cliente com:
  - Nome completo  
  - E-mail  
  - Idade  
  - Cidade  
  - Sexo  
  - Observações  
- Validação de campos obrigatórios  
- Verificação se idade é numérica  
- Página de confirmação com os dados enviados  

## 💻 Tecnologias
- HTML5  
- CSS3  
- PHP  

## ▶️ Como usar
1. Coloque os arquivos na pasta do servidor (XAMPP/WAMP)  
2. Abra o navegador em:
   http://localhost/SistemaCadastroClientePHP  
3. Preencha o formulário e envie
